# Portfolio GitHub Pages

Template portfolio pribadi berbasis HTML, CSS, dan JavaScript. Cocok untuk dipublikasikan sebagai GitHub Pages.

## Cara Pakai

1. Ganti teks `Nama Kamu`, email, username GitHub, dan link LinkedIn di `index.html`.
2. Ganti daftar proyek, skill, dan pengalaman sesuai profil kamu.
3. Buat repository baru di GitHub, misalnya `portfolio`.
4. Upload semua file ini ke repository tersebut.
5. Buka `Settings` > `Pages`, lalu pilih branch `main` dan folder `/root`.

Website akan tersedia di:

```text
https://username.github.io/portfolio/
```

Kalau repository diberi nama `username.github.io`, alamatnya akan menjadi:

```text
https://username.github.io/
```
